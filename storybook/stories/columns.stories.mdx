# Columns

A simple flexbox grid system can be leveraged through the `c4-columns` component.

Whether or not the grid has a gutter can be controlled with the `has-gutter`
property.

By default, direct children of `c4-columns` will render as columns and stretch
to share available space evenly. These children can optionally receive a number
of classes to hard code their width on large screens:

- `column--one-fourth`
- `column--one-half`
- `column--three-fourths`
- `column--one-third`
- `column--two-thirds`

On smaller screens these classes will not apply.
