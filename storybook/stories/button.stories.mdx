# Buttons

Our `c4-button` component can be used to display a styled `<button`> or `<a>`
tag. It has a number of options.

<Preview>
  <Story name="Button">
    {() => {
      const slot = text("Slot Content", "Button");
      const disabled = boolean("Disabled", false);
      const tag = select("Tag", ["button", "a"], "button");
      const type = select("Type", ["button", "reset", "submit"], "button");
      const href = text("href", "#");
      const buttonClass = select(
        "Modifier Class",
        ["", "secondary", "tertiary", "stretch", "left", "right"],
        ""
      );
      return html`
        <c4-button
          disabled="${disabled}"
          tag="${tag}"
          type="${type}"
          href="${href}"
          button-class="${buttonClass}"
        >
          ${slot}
        </c4-button>
      `;
    }}
  </Story>
</Preview>

## Disabled Buttons

Disabled buttons display with a faded background and disabled cursor. Only
`<button>` tags can be disabled, not `<a>` tags.

<Preview>
  <Story name="Disabled">
    {() => html`
      <c4-button disabled="true">
        Disabled Button
      </c4-button>
    `}
  </Story>
</Preview>

## Secondary and Tertiary Buttons

Secondary and Tertiary buttons can be used to communicate actions outside of the
primary flow.

<Preview>
  <Story name="Secondary and Tertiary">
    {() => html`
      <c4-button>
        Primary Button
      </c4-button>
      <c4-button button-class="secondary">
        Secondary Button
      </c4-button>
      <c4-button button-class="tertiary">
        Tertiary Button
      </c4-button>
    `}
  </Story>
</Preview>

## Stretch Buttons

Stretch buttons will stretch to fill available width. These are useful for
primary calls to action.

<Preview>
  <Story name="Stretch">
    {() => html`
      <c4-button button-class="stretch">
        Stretch Button
      </c4-button>
    `}
  </Story>
</Preview>



## Left and Right Buttons

Buttons can be used to compose more complex button groups. Add the left or
right class to only round corners on one side of the button.

<Preview>
  <Story name="Left and Right">
    {() => html`
      <c4-button button-class="left">
        Left Button
      </c4-button>
      <c4-button button-class="right">
        Right Button
      </c4-button>
    `}
  </Story>
</Preview>
