C4 Stencil Components

This is a playground to explore stencil components. These components can then be used in a variety of environments:

  • Storybook (Documentation)
  • Storyblok (CMS Preview)
  • Framer (Design Tool)
  • Vanilla JS (Express)
  • React
  • Vue
  • Angular

There are a few reasons we built these using web components:

  1. Standards Based
  2. Browser Native
  3. Portability
